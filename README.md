# מחשבון בליסטיקה חיצונית — 5.56×45 נאטו

מחשבון בליסטי לקליבר 5.56×45 נאטו, מבוסס על מודל G1 בתנאי ISA (15°C Sea Level).
זוהי גרסת ה־HTML/CSS/JS של הגיליון `מחשבון בליסטי סופי.xlsx`.

## תכולה

- בחירת נשק (12 קונפיגורציות) ותחמושת (6 קליעים)
- חישוב אוטומטי של מהירות פיה לפי אורך הקנה (ניתן לדריסה ידנית)
- הגדרת גובה כוונת ומרחק אפס (זירו)
- טבלת בליסטיקה מ־0 עד 1000 מ' בקפיצות של 25 מ': זמן עוף, מהירות, אנרגיה, נפילה טהורה, מסלול קליע מעל/מתחת לקו הכוון ו־MOA

## הפעלה

פותחים את `index.html` בכל דפדפן מודרני. אין תלויות חיצוניות.

```bash
open index.html      # macOS
xdg-open index.html  # Linux
```

## מבנה

| קובץ        | תיאור                                                         |
|-------------|----------------------------------------------------------------|
| `index.html`| המבנה (RTL עברית) + הקישורים ל־CSS/JS                          |
| `styles.css`| עיצוב כהה עם הדגשי צבע ירוק/אדום למסלול הקליע                  |
| `app.js`    | לוגיקת חישוב המסלול והסטייה (MOA), עדכון אוטומטי של הטבלה      |
| `data.js`   | מטריצת הנתונים המחושבים מראש (12 נשקים × 6 קליעים × 41 מרחקים) |

## נוסחאות מרכזיות

- **מסלול קליע (ס"מ)**: `−sight + range × (sight + drop@zero) / zero − drop(range)`
- **MOA**: `path × 100 / (2.908 × range)`  (1 MOA = 2.908 ס"מ ל־100 מ')
- **שינוי קנה**: ~9 m/s לכל אינץ' פחות מ־20"
- **שינוי מהירות פיה ידני**: האנרגיה מתעדכנת פרופורציונלית ל־v² והמהירות נמתחת לינארית
